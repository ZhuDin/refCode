1.3 浏览器战争

1.3.1 DHTML

    DHTML是"Dynamic HTML"（动态HTML）的简称。DHTML并不是一项新技术，而是描述
HTML、CSS和JavaScript技术组合的术语。其背后的含义是：

    ·利用HTML把网页标记为各种元素；

    ·利用CSS设置元素样式和它们的显示位置；

    ·利用JavaScript实时地操控页面和改变样式。
