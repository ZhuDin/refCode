Python内置的主要对象类型有：数、字符串、列表、元组和字典。

-------------------------------------------------------------------------------
7.1 对象魔法

在面向对象编程中，术语对象大致意味着一系列数据（属性）以及一套访问和操作这些数据的
方法。使用对象而非全局变量和函数的原因主要有：

    多态：可对不同类型的对象执行相同的操作。
    
    封闭：对外部隐藏有关对象工作原理的细节。
    
    继承：可基于通用类创建出专用类。
